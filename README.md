# 我的餐廳清單
利用node.js與mongoDB資料庫打造的餐廳清單，可檢視喜愛餐廳的詳細資料，也可新增或刪除喜愛的餐廳清單。

## 網站頁面
![restaurant_ver 2](https://user-images.githubusercontent.com/118960946/209093789-ab2cfab0-af6f-412f-9209-181eaff7b5db.png)


## 功能介紹
● 查看所有餐廳

● 瀏覽餐廳的詳細資訊

● 連結餐廳的地址到 Google 地圖

● 搜尋特定餐廳

● 新增餐廳

● 編輯餐廳

● 刪除餐廳

## 使用說明
1. 請先確認有安裝 node.js 與 npm

2. 將專案 clone 到本地

3. 在本地開啟之後，透過終端機進入資料夾，輸入：
```
npm install
```
安裝完畢後，繼續輸入：

```
npm run start
```
若看見此行訊息則代表順利運行，打開瀏覽器進入到以下網址
```
http://localhost:3000
```
若欲暫停使用
```
ctrl + c
```
